Under the Hood

Built for Musicianship

Every instrument is powered by the Web Audio API — low latency, high fidelity, and no installation required.

Keyboard Mapped

Every instrument maps to your QWERTY keyboard. Play naturally without a MIDI controller or audio interface.

🎛️
Real-Time Controls

Volume, reverb, sustain, octave shifting, and waveform selection — all adjustable mid-performance without interruption.

🔊
Web Audio Engine

Built on the Web Audio API with multi-oscillator synthesis, convolution reverb, and precise gain envelope shaping.

📱
Touch Enabled

Multi-touch support lets you play chords and gestures on phones and tablets with the same responsiveness as desktop.
